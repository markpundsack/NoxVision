# Security Policy

## Supported Versions

The following versions of NoxVision are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow these steps:

1. **How to report:** Send a detailed report via email to **marc.grox22@gmail.com**. Please do **not** open public GitHub issues for security vulnerabilities to protect all users.
2. **Response time:** You can expect an initial acknowledgment of your report within **48 hours**.
3. **Update frequency:** Once a vulnerability is accepted, you will receive status updates every **3 to 5 days** until the issue is resolved.
4. **What to expect:** - **Accepted:** We will work on a fix and release it in a future update (e.g., via the Google Play rollout).
   - **Declined:** You will receive a detailed explanation if the report is not classified as a security risk.
