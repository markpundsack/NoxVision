# NoxVision Closed Alpha Test

This page explains how external testers can join the Google Play closed alpha.

## Tester links

- Join tester group: `https://groups.google.com/g/noxvision-closed-testers`
- Opt-in on web: `https://play.google.com/apps/testing/com.noxvision.app`
- Install/update app: `https://play.google.com/store/apps/details?id=com.noxvision.app`
- Report feedback: `https://github.com/nacl-dev/NoxVision/issues`

## Tester steps

1. Join the Google Group.
2. Open the web opt-in link with the same Google account.
3. Accept the test invitation.
4. Install or update via the Play Store link.
5. Report bugs with device model, Android version, app version, and reproduction steps.
